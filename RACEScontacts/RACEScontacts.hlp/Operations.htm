RACES Contact List Operations

The Toolbar

Open and Load Database

Opens the database and displays table information.

Sort Contacts

A drop down menu to sort the Contact List by name (last name, first name) or by FCC call sign. Defaults to Sort by Name. Displays the list.

Sort by Name

Sort by Name (last name, first name) and displays the list.

Sort by Call Sign

Sort by FCC call sign and displays the list.

Save File

Displays a Save File Dialog and saves the currently displayed sorted list to the file selected as a "txt" (i.e. text file). Note, the default names are chosen by the current sort and it remembers the last output directory.

Sort and Save cSV File

A drop down menu to sort the Contact List by name (last name, first name) or by FCC call sign and then saves the data as a Comma Separated Value (CSV) file. Defaults to Sort By Name. Note, the default names are chosen by the sort chosen and it remembers the last output directory. The csv file is not displayed in the window of the application and is not affected by the currently displayed list.

Sort by Name

Sorts by Name and saves the data as a CSV file.

Sort by Call Sign

Sorts by FCC Call Sign and saves the data as a CSV file.

Print

Will do all the normal things to print the currently displayed sorted list.

Options

The options dialog allows specifying the Display and Printer Font Size Scale Factor, the printer orientation and the printer margins.

About

Displays the About Box. The about box uses text from the resource file (TestApp.rc) for the output information. There is a standard about box dialog in the Library.

Exit

Exit the application with no reservations.

Menus

Most of the menu items correspond to the toolbar descriptions. Only the extra menu items are described here.

File Menu

Print Preview

In a new window displays the currently displayed sorted list as the list would appear when printed.

Print Setup

The printer may be chosen. Note the orientation of the printer output may also be managed in the Options dialog.

Help Menu

Introduction

Starts the Help application at the Introduction page.
