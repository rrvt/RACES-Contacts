RACES Contact List

Introduction

This application will load the tables from the RACES database. Then there are various commands to output the contact information in various forms:

  • Send the data to a printer (or pdf printer)
  • Send the data to a ".txt" file
  • Send the data to a comma separated value file suitable for loading into Excel

The outputs may be sorted by member namd, last name first. The outputs may be sorted by the FCC Call Sign.
